Proprietatea display: inline, inline-block si block

Proprietatea de afișare CSS stabilește dacă un element este tratat ca element bloc sau element inline;

Comparativ cu display: inline, diferența majoră este că display: inline-block permite setarea unei lățimi și înălțimi a elementului.
De asemenea, cu display: inline-block, marginile/padding-urile de sus și de jos sunt respectate, dar cu display:inline nu.
