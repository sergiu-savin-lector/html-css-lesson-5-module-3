Model de element bloc (margin, padding)

Text
Content (zona de conținut)
conținutul unui element, cum ar fi un text, o imagine, un videoclip sau elemente imbricate. În mod implicit, înălțimea zonei de conținut este determinată de conținut, lățimea este determinată de tipul elementului (bloc sau inline).
Padding (umplutura)
distanța dintre conținutul unui element și bordura acestuia. Folosit pentru efecte decorative. Nu poate avea valori negative.
Border (bordura)
chenarul elementului, în mod implicit lățimea sa este zero. Dacă nu este specificată nicio culoare a bordurei, aceasta capătă culoarea conținutului principal, cum ar fi textul.
Margin (marginea)
adaugă o margine înafara elementului, de la marginea exterioară (bordură) la elementele adiacente, separându-le astfel pe pagină. Poate avea valori negative.
